개요
---
실시간 영상 처리 기술을 활용하여 운전자의 졸음 여부를 판펼하고 졸음운전 중이라면 시각 신호와 음성 신호의 경고가 울리게 하는 프로그램을 제작하였습니다. opencv와 dlib라이브러리를 활용하여 영상처리를 하였고, 야간과 주간 상황에 맞는 필터를 활용하여 더욱 안정적으로 작동하게 구현하였습니다.
